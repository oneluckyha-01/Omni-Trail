# 更新日志

## v1.2.1 (2026-06-02)

### iPad 触摸交互优化
- **分栏线触摸拖拽**：笔记、日历、图片嗅探三个面板的分栏线现已支持触摸拖拽调整宽度，与桌面端鼠标拖拽行为一致
- **触摸穿透修复**：面板内触摸滑动不再穿透到下层页面，通过 `touch-action` CSS 属性限制触摸事件传播

### 布局修复
- **快速记录面板**：候选集数列表移至独立行显示，不再与「总集数」输入框同行排列
- **日程管理年视图**：年视图切换时自动清除宽度限制，日历区域占满全宽；月视图恢复 `max-width: 400px` 约束
- **日程管理标题**：年预览视图标题正确显示「2026年」，不再出现多余的「月」字
- **日程管理最大化**：左侧日历区域添加 `max-width: 420px` 限制，大屏下保持合理宽度
- **笔记便签分栏线**：修复鼠标无法拖拽的问题，将可点击热区从 0.5px 扩大至 6px（采用与日历分栏线相同的伪元素方案）
- **笔记便签横线对齐**：搜索栏与编辑器标题的分隔线现在水平对齐（统一 `min-height: 42px`）

### 时钟工具修复
- **优酷页面时钟显示**：修复时钟/秒表在优酷页面文字重叠问题，添加显式 `line-height` 和 `white-space: nowrap`，防止宿主页面样式继承导致的挤压
- **时钟 `clamp` 修正**：修正 `font-size: clamp()` 参数顺序（min 值不应大于 max 值）
- **时钟字号隔离**：`.timer-wrap` 添加 `font-size: 16px`，防止宿主页面字号继承影响 `em` 单位计算
- **倒计时按钮默认状态**：默认聚焦按钮从 25 分钟修正为 5 分钟，与实际默认时长 `duration: 300` 一致

### 视觉优化
- **日历左侧背景**：已恢复原状（移除临时添加的背景色）

### 代码质量
- **分栏线 CSS 合并**：`.notes-resizer` 和 `.cal-resizer` 合并为同一组选择器，消除重复声明
- **冗余代码清理**：移除 3 处 `border-right: none` 冗余声明、5 处 `calc(var(--font-size) * 1.0)` 简化为 `var(--font-size)`、移除默认值 `max-width: none` 和无用 class 名
- **事件监听器优化**：分栏线拖拽改为按需绑定/解绑 `window` 事件，避免永久挂载导致的监听器累积

### 已知限制
- Safari 标签栏遮挡问题需修改宿主页面 viewport meta 标签，油猴脚本层面暂无法解决
