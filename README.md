# 万象 · Omni Trail

浏览器悬浮球全能助手 — 一个集追剧管理、网页导航、Markdown 笔记、日程待办、时钟工具、图片嗅探、视频解析于一体的 Tampermonkey 用户脚本。

## 功能特性

- **追剧进度管理** — 记录观看进度、评分、状态追踪，支持 Bangumi 元数据自动刮削
- **观影平台管理** — 自定义常用平台分类与快捷跳转
- **网页导航收藏** — 多引擎搜索 + 自定义书签分类
- **Markdown 笔记** — 内置编辑器，支持实时预览
- **日程管理** — 待办事项 + 日期标注，支持农历、重复规则（公历/农历）、自动顺延
- **时钟工具** — 实时时钟 / 秒表（计次） / 倒计时 / 番茄钟，每日一言
- **视频解析** — 多源解析播放，支持页面注入模式
- **图片嗅探** — 一键提取页面图片，按尺寸/格式过滤
- **WebDAV 云同步** — 坚果云等 WebDAV 服务备份与恢复
- **18 套主题** — 毛玻璃 / 深色 / 浅色多风格，支持自定义字体与字号

## 安装

### 前置要求

安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展。

### 安装脚本

1. 点击 Tampermonkey 图标 → 添加新脚本
2. 粘贴 `omni-trail.js` 的全部内容
3. 保存（Ctrl+S）

或直接访问脚本文件 URL，Tampermonkey 会自动弹出安装提示。

## 使用

安装后任意页面右下角出现悬浮球，点击展开主面板。各功能模块通过顶部标签页切换。

| 快捷键 | 功能 |
|--------|------|
| `Shift+E` | 打开主面板 |
| `Shift+R` | 快速记录（追剧页面） |
| `Shift+G` | 观影平台 |
| `Shift+D` | 网页导航 |
| `Shift+N` | 笔记便签 |

快捷键可在设置中自定义。

## 数据存储

所有数据存储在浏览器本地（Tampermonkey 存储），支持导出 JSON 备份文件或通过 WebDAV 同步到云端。

## 兼容性

- Chrome / Edge / Firefox（需 Tampermonkey 或 Violentmonkey）
- 匹配所有网站（`*://*/*`）
